Vítejte na burze učebnic!

Tato stránka slouží jako uživatelská nápověda a návod jak používat tuto aplikaci.

Přihlašování

Do aplikace se hlásí s pomocí Microsoft Office 365 účtu (vašim školním účtem). Bude pravděpodobně z domény eskola.eu. Heslo pro účet je také stejné.

Níže jsou přesné kroky pro přihlášení:

  1. Na přihlašovací stránce aplikace klikněte na "Přihlásit". Budete přesměrováni na stránky Microsoft.
  2. Zde se přihlašte jako do jakékoli jiné aplikace (např. MS Teams). Při prvním přihlášení možná bude třeba dát aplikaci jistá oprávnění.
  3. Po přihlášení a odsouhlasení budete přesměrováni zpět do aplikace. Aplikaci může pár sekund trvat, než ověří vaše přihlášení

Hlavní stránka

Po přihlášení jste přesunuti na hlavní stránku aplikace. Zde máte přístup k několika funkcionalitám:

  1. Přepínání jazyka
  2. Přepínání mezi světlým a tmavým režimem
  3. Zobrazení okna příchozích zpráv
  4. Uživatelské menu (další funkcionality)
  5. Filtr zobrazených příspěvků
  6. Styl řazení příspěvků
  7. Tlačítko pro načítání dalších příspěvků
  8. Tlačítko pro přidání nového příspěvku

Přispěvky

Příspěvky slouží jako inzeráty oznamující o prodeji použité učebnice. Součástí takového příspěvku je:

  • Titulek - Výstižný
  • Doba trvání - Jak dlouho bude inzerát na stránce
  • Předměty - Výběr předmětů, pro které je učebnice určena
  • Stav učebnice - V jaké je učebnice přibližné kvalitě
  • Ročníky - V jakých ročnících se učebnice používá
  • Cena - Buď stacionární cena, nebo rozsah
  • Fotky - Ilustrační obrázky vaší učebnice
  • Kontakt - U příspěvku je jméno tvůrce a možnost ho kontaktovat
    • Buď přes chat v MS Teams (Tlačítko Kontaktovat)
    • Nebo přes jiné kanály (Tlačítko tři tečky)

Správa účtu

Pro správu svého účtu otevřete menu a zvolte Můj účet. Na této stránce má uživatel k dispozici dvě hlavní možnosti:

  1. Upravit své kontaktní informace
  2. Smazat svůj účet (a všechny vaše příspěvky)

Moje příspěvky

Uživatel má možnost spravovat své vlastní příspěvky. Na danou stránku se opět dostane přes menu, tlačítko Moje příspěvky. Jsou zde možné následující úkony:

  1. Prodloužit trvanlivost - Když chce uživatel prodloužit dobu trvání svého inzerátu
  2. Přidat info - Pokud by chtěl uživatel přidat další informace (dodatečné informace, změny ve stavu)
  3. Odebrat - Odebrat příspěvek (např. již prodané knihy)

Administrační rozhraní

Administrátorem aplikace se v základu stává vždy první uživatel. Tento první administrátor pak určuje další administrátory aplikace.

Domovská stránka

Domovská stránka se pro administrátora příliš neliší od řadového uživatele. Existuje zde však možnost odebírat nevhodné příspěvky.

Pro odebrání příspěvku:

  1. U příspěvku rozklikněte tři tečky
  2. Zvolte Odebrat
  3. Potvrďte

Také uživatelské menu uživatele je pro administrátora jiné, obsahuje následující

  • Uživatelé - Správa účtů v aplikaci
  • Blacklist - Správa permanentně zakázaných uživatelů
  • Předměty - Vyučovací předměty
  • Poslat zprávu - Rozesílání globálních zpráv

Uživatelé

Na této stránce se s uživateli dají provádět následující úkony:

  • Smazat příspěvky - Smaže všechny příspěvky uživatele
  • Smazat účet - Smaže příspěvky i všechna ostatní uživatelská data jako Kontakty, apod.
  • Zakázat přístup - Dočasný zákaz uživatele na platformě
  • Historie zákazů - Historie všech zákazů, které uživatel obdržel
  • Blacklist - Smaže uživatelská účet a permanentně zakáže uživateli přístup na platformu. (Používat jen ze závažných důvodů)
  • Změnit na ... - Možnost měnit roli uživatele (přidat, či odebrat administrátorská oprávnění)

Blacklist

Tato stránka podává krátký přehled o permanentně zakázaných uživatelích. Zákaz je vytvořen na Microsoft Id jejich účtu.

Předměty

Stránka pro správu vyučovacích předmětů. Je zde možnost předměty přidávat i odebírat. Funkcionalita slouží hlavně k tomu, aby uživatel mohl daný předmět specifikovat při tvorbě příspěvku.

Poslat zprávu

Rozhraní pro posílání globálních zpráv.

Zprávy mohou být pro:

  • Všechny uživatele
  • Všechny administrátory

Welcome to the Textbook Marketplace!

This page serves as a user guide on how to use this application.

Login

You log into the application using your Microsoft Office 365 account (your school account). It will most likely be from the eskola.eu domain. The password for the account is the same.

Below are the exact steps to log in:

  1. On the application's login page, click "Login". You will be redirected to Microsoft's website.
  2. Log in as you would to any other application (e.g., MS Teams). During your first login, you may need to grant the application certain permissions.
  3. After signing in and confirming, you will be redirected back to the application. It may take a few seconds for the application to verify your login.

Homepage

After logging in, you are taken to the homepage of the application. Here you have access to several features:

  1. Language switch - Switch to en
  2. Switch between light and dark mode
  3. Open the incoming messages window
  4. User menu (additional features)
  5. Filter displayed posts
  6. Sort posts by
  7. Button to load more posts
  8. Button to add a new post

Posts

Posts serve as advertisements announcing the sale of a used textbook. Such a post includes:

  • Title - Concise and descriptive
  • Duration - How long the advertisement will remain on the page
  • Subjects - Selection of subjects the textbook is intended for
  • Condition - Approximate quality of the textbook
  • Grades - Which grade levels use the textbook
  • Price - Either a fixed price or a range
  • Photos - Illustrative images of your textbook
  • Contact - The post includes the creator's name and the option to contact them
    • Either via chat in MS Teams (Contact button)
    • Or through other channels (three dots button)

Account Management

To manage your account, open the menu and select My Account. On this page, the user has two main options:

  1. Edit contact information
  2. Delete your account (and all your posts)

My Posts

The user has the option to manage their own posts. You can access this page again through the menu, using the My Posts button. The following actions are available:

  1. Extend duration - When the user wants to extend the duration of their advertisement
  2. Add info - If the user wants to add additional information (extra details, changes in condition)
  3. Remove - Remove the post (e.g., book's already sold)

Administration Interface

The first user of the application automatically becomes an administrator by default. This first administrator then appoints other administrators.

Home Page

The home page for an administrator does not differ much from that of a regular user. However, there is an option to remove inappropriate posts.

To remove a post:

  1. Click the three dots on the post
  2. Select Remove
  3. Confirm

The user menu is also different for administrators and includes the following:

  • Users - Manage accounts in the application
  • Blacklist - Manage permanently banned users
  • Subjects - School subjects
  • Send Message - Send global messages

Users

On this page, the following actions can be performed with users:

  • Delete posts - Deletes all posts of the user
  • Delete account - Deletes posts and all other user data such as contacts, etc.
  • Restrict access - Temporary ban of the user from the platform
  • Ban history - History of all bans the user has received
  • Blacklist - Deletes the user account and permanently bans the user from the platform (Use only for serious reasons)
  • Change to ... - Option to change the user's role (grant or remove administrator privileges)

Blacklist

This page provides a brief overview of permanently banned users. The ban is created based on the Microsoft ID of their account.

Subjects

Page for managing school subjects. Subjects can be added or removed here. This functionality mainly allows users to specify a subject when creating a post.

Send Message

Interface for sending global messages.

Messages can be sent to:

  • All users
  • All administrators
