Posts

Operations about posts

Retrieve a list of 10 posts

Retrieve a list of 10 posts ordered and filtered by criteria

Authorizations:
(ssidCookiecsrfHeader)
query Parameters
begin
required
integer

Skip first X posts

orderBy
required
string
Enum: "date" "price"
Example: orderBy=date

Order the posts by most recent or lowest price

filterSubjects
required
Array of strings
Example: filterSubjects=ANJ,PSI

All subjects tags of shown posts (logical OR) (When no filter, specify ALL)

filterYears
required
Array of integers
Example: filterYears=1,3

All year tags of shown posts (logical OR) (When no filter, specify ALL)

filterState
required
Array of strings (state)
Items Enum: "Like new" "Good" "Worn"
Example: filterState=Good,Worn

All state tags of shown posts (logical OR) (When no filter, specify ALL)

priceMin
number
Default: 0
Example: priceMin=100

Lower end of price range

priceMax
number
Default: "MAX_PRICE"
Example: priceMax=300

Upper end of price range

Responses

http://localhost/posts
